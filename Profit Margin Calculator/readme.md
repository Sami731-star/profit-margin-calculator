# Advanced Profit Margin Calculator

A professional web-based calculator that includes all real business costs.

## Features
- Product cost
- Advertisement cost
- Shipping cost
- Total cost calculation
- Net profit or loss
- Profit margin & markup

## Formula Used
- Total Cost = Product + Ads + Shipping
- Profit = Selling Price − Total Cost
- Margin = (Profit / Selling Price) × 100
- Markup = (Profit / Total Cost) × 100

## Built With
- HTML5
- CSS3
- JavaScript
